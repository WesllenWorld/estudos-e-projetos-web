Trabalhando com as ibagens

Tipo bitmap e tipo vetorial

Bitmap: é uma imagem formada por pixels, ou seja, pontos. Cada ponto é uma cor. Quanto maior a imagem, maior a quantidade de pontos e maior o tamanho do arquivo.

Vetorial: é uma imagem formada por vetores, ou seja, linhas. Cada linha é uma cor. Quanto maior a imagem, maior a quantidade de linhas e maior o tamanho do arquivo (perceba que ao dar zoom na página, esta imagem não perde qualidade).

Imagens transparentes

São imagens com fundo transparente. O primeiro é um gif e o segundo é um png.

GIF

É um formato de imagem que suporta animações. É muito utilizado em banners.

Float

É uma propriedade do CSS que permite que um elemento fique ao lado do outro.
